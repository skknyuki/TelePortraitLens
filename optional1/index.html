    撮影クオリティ設定

    デフォルトでは、撮影クオリティは 4K に設定されています。
    解像度を変更したい場合は、RenderTexture の Size を変更してください。

    RenderTexture は、以下のフォルダ内にあります。

    [GoatCannery]/[TelePortraitLens]/[Textures]/[RenderTexture]

    各 RenderTexture の役割

    • "TPL_Main"
      メインの被写体を撮影するカメラです。
      この解像度を、保存したい写真の解像度に合わせて設定してください。

    • それ以外のテクスチャ "TPL_Back", "TPL_Fore", "TPL_Fore_depth"
      これらは "TPL_Main" の半分の解像度 に設定してください。

    推奨解像度

    テクスチャ名 フルHD QHD 4K（デフォルト） 8K
    TPL_Main 1920x1080 2560x1440 3840x2160 7680x4320
    それ以外 960x540 1280x720 1920x1080 3840x2160

    Anti-aliasing について

    "TPL_Main" の Anti-aliasing を上げると、写真の品質は向上します。
    ただし、その分 VRChat の負荷も大きくなる ため、
    自身のPCスペックと相談しながら設定してください。
    初期設定は [2 samples]になっております。

    補足

    "TPL_Back", "TPL_Fore", "TPL_Fore_depth" の解像度を上げても、
    内部では 半分の解像度で処理 し、最終的にボケさせるため、
    必要以上に高い解像度へ設定することはおすすめしません。

    注
    HD(720P)サイズには対応しておりません。
